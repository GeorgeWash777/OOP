чтобы запустить лабу билдим из папки выше папки lab5:
```
mkdir -p build
cd build
cmake ..
make
```

в файле CMakeLists.txt (выше папки лабы) коментим не нужные лабы

в консоли из папки build выполняем команду, чтобы запустить лабу:
```
./lab5/lab5_exe
```
